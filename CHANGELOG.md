# Changelog

All notable changes to this project will be documented here.

## 0.1.0-demo.0 - 2026-06-25

Initial public demo release.

- Reworked the project from a Vite template into a Joseon zombie visual-novel demo.
- Added script-driven VN playback with labels, jumps, choices, variables, scenes, effects, and character commands.
- Added cinematic presentation: intro overlay, letterboxing, transitions, snow, flash, and shake effects.
- Added generated demo backgrounds, sprites, dialogue portraits, and concept contact sheets.
- Added opening procedural `windy_snow` BGM through Web Audio.
- Added README, changelog, license, and public package metadata for GitHub release readiness.
